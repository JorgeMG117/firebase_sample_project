Deploying a Static Website to Firebase Hosting

Firebase Hosting is a fast, secure, and free (for moderate usage) platform for deploying static websites — HTML, CSS, JS, and static assets like images or JSON.

🌐 Global CDN
🔒 Free SSL
⚡ Lightning Fast
💰 Free Tier

Prerequisites

Before starting, make sure you have:

📦

Node.js and npm

Check with:

node -v
npm -v

If not installed, download from nodejs.org

🔑

Firebase Account

Sign up or log in at firebase.google.com

🏗️

Google Project

You can create one during the Firebase setup process

What are Node.js and npm?

  • Node.js is a JavaScript runtime that allows you to run JavaScript outside the browser — commonly used for building and managing web applications.
  • npm (Node Package Manager) is a tool that comes with Node.js and lets you install and manage JavaScript packages (like Firebase tools).

Step-by-Step Guide

1

Install the Firebase CLI

Install the Firebase Command Line Interface globally:

npm install -g firebase-tools

Check installation:

firebase --version
2

Login to Firebase

Authenticate your CLI with your Google account:

firebase login
3

Initialize Your Project

Navigate to your website's folder and initialize Firebase Hosting:

cd path/to/your/project
firebase init

Follow the prompts:

  1. Select Hosting: Set up deployments for static web apps
  2. Create a Firebase project
  3. Set the public directory (e.g., public)
  4. Say no to single-page app
  5. Skip GitHub deploys for now
4

Add Your Website Files

Ensure your site files are in the chosen public directory:

my-website/
├── public/
│   ├── index.html
│   ├── style.css
│   ├── script.js
├── firebase.json
└── .firebaserc
5

Preview Locally

firebase serve

or

firebase emulators:start

Visit http://localhost:5000 to preview.

6

Deploy to Firebase

firebase deploy

Output example:

✔ Deploy complete!

Project Console: https://console.firebase.google.com/project/myproject
Hosting URL: https://myproject.web.app
7

Set Up a Custom Domain (Optional)

  1. Go to Firebase Console → Hosting
  2. Click "Add custom domain"
  3. Follow DNS setup instructions
  4. Firebase provides free SSL automatically

What is Firebase?

Firebase is a comprehensive app development platform by Google that provides a suite of tools and services to help developers build, improve, and grow their applications.

Firebase Services

🌐

Firebase Hosting

Fast and secure web hosting for static content with global CDN

🔥

Firebase Realtime Database

Cloud-hosted NoSQL database that syncs data in real-time

🔐

Firebase Authentication

Easy-to-use authentication system with multiple providers

☁️

Firebase Cloud Functions

Serverless functions that run in response to events

📊

Firebase Analytics

Free app measurement and user engagement analytics

🚀

Firebase Performance

Monitor app performance and identify bottlenecks

Why Choose Firebase Hosting?

🌍 Global CDN: Content delivered from edge locations worldwide for maximum speed
🔒 Free SSL: Automatic SSL certificates with automatic renewal
⚡ Fast Deploy: Deploy with a single command using Firebase CLI
🎯 Custom Domains: Easy DNS configuration for your own domain
📊 Analytics: Built-in analytics to track performance and usage
🔄 Version Control: Rollback to previous versions instantly

Common Commands

firebase init
Initialize Firebase in your project
firebase serve
Preview locally
firebase deploy
Deploy your site
firebase login
Log into Firebase
firebase logout
Log out of Firebase
firebase open hosting:site
Open your live site

Troubleshooting

Error: Hosting setup requires a public directory

→ Ensure your firebase.json points to the right folder.

Website not updating

→ Try firebase deploy --only hosting

Permission denied

→ Log into the correct Google account.
