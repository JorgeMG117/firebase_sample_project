Step-by-Step Guide

1

Install the Firebase CLI

Install the Firebase Command Line Interface globally:

npm install -g firebase-tools

Check installation:

firebase --version
2

Login to Firebase

Authenticate your CLI with your Google account:

firebase login
3

Initialize Your Project

Navigate to your website's folder and initialize Firebase Hosting:

cd path/to/your/project
firebase init

Follow the prompts:

  1. Select Hosting: Set up deployments for static web apps
  2. Create a Firebase project
  3. Set the public directory (e.g., public)
  4. Say no to single-page app
  5. Skip GitHub deploys for now
4

Add Your Website Files

Ensure your site files are in the chosen public directory:
